# 📂 Advanced AI File Organizer
A high-performance, web-based tool to instantly sort messy folders using concurrent processing.

## ✨ Features
- **Parallel Processing**: Uses multi-core threading to sort hundreds of files in seconds.
- **Auto-Cleanup Engine**: Automatically wipes temporary workspaces to prevent server storage errors.
- **Smart Categorization**: Sorts files into specialized folders for PDF, Word, Video, and more.
- **Multi-User Stable**: Uses unique UUID workspaces to prevent file-access conflicts.

## 🛠️ Tech Stack
- **Language**: Python
- **Frontend**: Streamlit
- **Deployment**: Docker & Hugging Face Spaces
